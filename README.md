* Fill configuration (config.json):
```json
{
    "token": "make a secret called token and value paste the token"
    "prefix": "g!",
    "everyoneMention": true,
    "hostedBy": true
}
```

* Install dependencies:
```sh
npm install
```

* Start:
```sh
node index.js
```
